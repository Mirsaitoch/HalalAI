# HalalAI iOS — локальная разработка

Краткая шпаргалка для запуска клиента. Полное описание модулей и архитектуры — в [`README.md`](./README.md); контекст для ассистентов и ревью — в [`docs/ios.md`](../docs/ios.md).

## Требования

- **Xcode** 15+ (в монорепо в `CLAUDE.md` указано 26.3+ при наличии такой среды)
- **iOS** 16+ (минимальная версия в `docs/ios.md`; целевой SDK задаётся в проекте)
- **Swift** 5.10+

## Открыть проект

```bash
open HalalAI-ios/HalalAI.xcodeproj
```

Из корня монорепо путь тот же: `HalalAI.xcodeproj` лежит внутри `HalalAI-ios/`.

## Сборка из терминала

```bash
xcodebuild -project HalalAI-ios/HalalAI.xcodeproj -scheme HalalAI -configuration Debug
```

Запуск тестов (unit):

```bash
xcodebuild -project HalalAI-ios/HalalAI.xcodeproj -scheme HalalAI -destination 'platform=iOS Simulator,name=iPhone 16' test
```

При необходимости замените симулятор на доступный в `xcrun simctl list`.

## Backend для отладки

Чат и авторизация идут на **Spring Boot**, а не на LLM-сервис напрямую. Поднимите стек из `HalalAI-backend/` (`docker compose up -d`) или только API по инструкции в [`HalalAI-backend/README.md`](../HalalAI-backend/README.md).

В **DEBUG** по умолчанию используется `http://localhost:8080` (см. `docs/ios.md`). Для HTTP к localhost в симуляторе в `Info.plist` должны быть настроены исключения домена (как в текущем проекте).
